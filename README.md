# AI Smart Learning Platform

## What does this project do?
Most quiz apps just tell a student if their answer is "right" or "wrong." This project is different because it acts like a real tutor. When a student answers a question, an AI checks it and gives a friendly explanation so they actually understand the topic.

It also helps teachers by showing them exactly which students are struggling and which topics are the hardest.

# Main Features
* **Student Chatbot:** Students take quizzes in an easy-to-use chat window.
* **Questions:** First, students answer questions typed by their teacher. When those run out, the AI automatically creates new practice questions.
* **Teacher Dashboard:** Teachers can securely log in to add new questions and view student grades.
* **Smart Analytics:** The database automatically calculates average scores and alerts teachers if a student's grade drops below 60%.

## Tools Used
* **Backend:** FastAPI (Python)
* **Frontend:** Streamlit (Python)
* **Database:** PostgreSQL (Aiven Cloud)
* **AI:** Groq AI
